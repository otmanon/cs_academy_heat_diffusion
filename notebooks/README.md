# Notebooks

Jupyter notebooks where students fill out and run code go here. Each notebook
pairs with the lessons in `lessons/`.

- **`heat_diffusion.ipynb`** — the main notebook. Run it locally after `uv sync`.
- **`heat_diffusion_colab.ipynb`** — a self-contained [Google Colab](https://colab.research.google.com/)
  version that runs in the browser with nothing to install. Its top cell has an
  "Open in Colab" badge.

Launch the local notebook server with:

```bash
uv run jupyter notebook
```
